# VUE 出一個電商網站

## 使用的技術/套件
- Vue CLI 3
- Vue router
- Vue axios(AJAX) 串接 API
- Vue-loading-overlay 【動畫載入效果】
- Vee-validate(驗證表單)
- Vue-i18n(支持多語言)
- RWD設計
- SCSS
- BS4
- Webpack
- Font awesome
- localStorage
